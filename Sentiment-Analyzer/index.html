Week 1 — Sentiment Analyzer (VADER NLP)

Goal: identify the most promising dormant IP for revival using community sentiment data.

The tool

Custom sentiment analysis pipeline built on VADER (Valence Aware Dictionary and sEntiment Reasoner), optimized for short, informal text like Reddit posts and comments.

Steps:

  • Collected Reddit posts/comments mentioning dormant titles.
  • Cleaned and normalized text for better sentiment accuracy.
  • Scored each entry using VADER’s compound score.
  • Aggregated by title and ranked using:
    • Positive sentiment %
    • Discussion volume
    • Trend velocity

Core script snippet:

from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer
analyzer = SentimentIntensityAnalyzer()

text = "Spec Ops remake would be amazing!"
score = analyzer.polarity_scores(text)
print(score['compound'])
Result 
Winner: Spec Ops
Reason: highest positive sentiment + strong sustained discussion + frequent remake requests

Why it matters

Instead of guessing which title to revive, this method turns community chatter into measurable, reproducible data — providing evidence for strategic decisions and investor discussions.
