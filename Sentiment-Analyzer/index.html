Week 1 — Sentiment Analyzer (VADER NLP)

Goal: find the most promising dormant IP for revival using community sentiment.

Approach

Built a sentiment analysis pipeline around VADER (Valence Aware Dictionary and sEntiment Reasoner), ideal for short, informal text like Reddit posts.

Steps:

  • Collected Reddit posts/comments mentioning dormant titles.
  • Cleaned and normalized text for consistent analysis.
  • Scored each using VADER’s compound sentiment score.
  • Ranked titles by:
    • Positive sentiment %
    • Discussion volume
    • Trend velocity

Core script snippet:

from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer

analyzer = SentimentIntensityAnalyzer()
text = "Spec Ops remake would be amazing!"
score = analyzer.polarity_scores(text)
print(score['compound'])

Result

Winner: Spec Ops
Reason: highest positive sentiment + sustained discussion + clear remake interest

Why it matters

This turns vague “buzz” into measurable, reproducible data, giving investors and decision-makers confidence in choosing a revival candidate.
