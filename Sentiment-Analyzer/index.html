Week 2 — AAA Game Pitch: Sentiment Analysis

Core research for a Triple-A remake investment pitch — using NLP to gauge community demand.

Project Context

Week 2 deliverable for training program

As part of preparing a AAA investment pitch, I needed to choose the most commercially promising dormant IP to remake. To support this decision, I built a sentiment analyzer using VADER NLP and ran it on Reddit discussions about several candidate titles.

Why Sentiment Analysis?

  • Quantifies positive, neutral, and negative sentiment from gaming discussions
  • Helps validate market demand beyond personal opinion
  • Supports investor confidence with measurable community data

Example Code (VADER NLP)

Minimal demo — actual analysis looped over thousands of Reddit posts

from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer

analyzer = SentimentIntensityAnalyzer()
text = "Spec Ops remake would be amazing!"
score = analyzer.polarity_scores(text)
print(score['compound'])  # 0.771 (positive sentiment)

Result

Spec Ops showed the highest share of positive sentiment and sustained discussion volume, making it the strongest candidate for the remake pitch. Investors liked the analysis approach, though noted the previous title’s sales history as a risk factor.
